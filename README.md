# SkillSwap
A modern Learning Management System (LMS) built with React and Vite.

## Overview
SkillSwap connects **Trainees** seeking knowledge with expert **Trainers**. Originally starting as a basic skill-swapping platform, the application has evolved into a robust course management system.

## Key Features

### 🎓 Trainee Experience
- **Browse Courses**: Discover and explore available courses.
- **Unlock Videos**: Pay enrollment fees to unlock high-quality video content.
- **Track Progress**: Easily monitor learning completion status.
- **Certificates**: Automatically earn and download certificates upon course completion.

### 🧑‍🏫 Trainer Experience
- **Create Courses**: Build rich course materials and upload video content.
- **Set Pricing**: Determine the appropriate fee for enrolled members.
- **Manage Portfolio**: Instantly view pending applications and approved published courses.
- **Trainee Reviews**: Track user reviews to improve content.

### 🛡️ Admin Experience
- **Manage Users**: Review pending Trainer applications, approve credentials, and view active users.
- **Approve Videos**: Supervise quality control and vet uploaded course materials.
- **Set Course Fees**: Configure base pricing and system commissions.
- **Payment Reports**: Track global revenue, trainee enrollments, and trainer payouts.

## Tech Stack
- Frontend Dashboard: React, React Router
- Build Tool: Vite
- Icons: Lucide React
- Routing: Client-side dynamic tab navigation & browser routing
- Styling: Custom Vanilla CSS (fully responsive desktop/mobile layouts)

## Getting Started

1. Clone the repository.
2. Navigate into the client directory:
   ```bash
   cd client
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## Workflow
All layout styling has been carefully refactored to prioritize desktop and modern mobile responsiveness dynamically, removing previously mobile-restricted static borders and layouts.
