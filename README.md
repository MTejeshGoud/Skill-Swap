# SkillSwap

SkillSwap is a community-driven platform designed to help people exchange skills, connect with local interest groups, and elevate their personal growth through peer-to-peer learning.

## Features

- **Skill Wallet**: Manage the skills you can teach and the ones you want to learn.
- **Circles**: Discover and join local communities like "Software Developers", "Foodies", or "Language Learners".
- **AI Matchmaker**: Find the perfect learning partner based on your interests using smart matching.
- **Trust Badges**: (Coming Soon) Build reputation and trust within the community.

## Technology Stack

- **Frontend**: React (Vite)
- **Styling**: Vanilla CSS (Custom Design System with CSS Variables)
- **Routing**: React Router DOM
- **Icons**: Lucide React

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build available for Production**
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components (Layout, etc.)
- `src/pages`: Individual screen views (Home, Profile, Circles, etc.)
- `src/index.css`: Global design system and variables.
